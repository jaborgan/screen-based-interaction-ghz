visual effect randomizer featuring 'face' app!

This is a prototype coded from scratch in p5.js using a sine wave effect to generate 'hair, ellipse function to simulate 'eyes' and an arc function to draw a mouth.

To interact with the application make sure the canvas is in focus ( click on the window)

Experiment pressing the keys H, E, F, S, and Space key! See screen for further instructions, Reload page if it gets too wild! Enjoy
